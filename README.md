# Profile-Optimizer-ChatBot
Profile Optimizer ChatBot is an interactive web-based AI assistant that helps users enhance their resumes, LinkedIn profiles, and overall career presence.
It provides actionable suggestions, analyzes uploaded text files (e.g., resume .txt), and gives structured feedback using AI through the OpenRouter API.

Features :

AI-Powered Chat System for career guidance

Resume/CV Improvement Tips with markdown-styled responses

LinkedIn Optimization Suggestions

.TXT File Upload Support for analyzing resume text

Rich UI/UX with Bootstrap styling, animations, and a draggable chat window

Client-side File Handling (no file stored on server)

Typing Indicator + Clean Chat Layout
